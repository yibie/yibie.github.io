如何在 M1 MacBook 上玩游戏

因为把 Switch 留在了深圳，我不得不开发在 MacBook 上玩游戏的技能。

Windows 游戏

直接下载 Steam 的 Mac 版，会失望的发现，很多好玩的游戏，平台只支持 Windows。所以，参考 2 条哔哩哔哩的视频教程，一步一步跟着走：

  • 【Mac 玩游戏】Crossover 21.1 教程 安装最新的 DXVK 玩 Windows 游戏

    阿婆主叫 js-john，是一名旅居日本的程序员，他还做了一个叫 iSign签名网 的网站，帮人安装 App Store 以外的 App，而且可以帮助苹果开发者分成。

  • 处之漠然的 CrossOver 教程合集

    阿婆主叫「处之漠然」，她的教程非常详细，如果担心自己的步骤出错了，可以对照着做。

要点

  • CrossOver 需付费购买，自行准备好钱钱钱。

  • CrossOver 要从海外官网购买和下载，地址在 CodeWeavers。

    国内版是海外版的阉割版，功能不全，为了避免意外发生，最好是下载海外版。

    由于点击官网的购买按钮，会跳转到中文购买页面，请自备梯子，并开启全局模式。

  • 必须更新视频中提到的 2 个插件：

    DXVK：https://github.com/marzent/dxvk/releases

    MoltenVK：https://github.com/Gcenx/MoltenVK/releases

    上不了 Github 的得自己想办法。

  • 关于游戏兼容性，可以查： https://www.applegamingwiki.com/wiki/Home

one more thing…

很神奇的是，用 Windows 虚拟机来玩游戏，要比 macOS 原生的流畅多了，发热也不严重。

只能说，大部分游戏对 macOS 的优化做得很差。

主机游戏

除了 PC 大作之外，也可以考虑主机游戏，很多经典神作让玩家沉浸在无与伦比的体验中，值得一再回味。比如我之前沉迷的《皇家骑士团 2》。

  • OpenEmu: 🕹 Retro video game emulation for macOS

    神器，支持的老主机特别多，如果是喜欢玩老游戏的，直接入它。缺点是不支持 PS2、PS3 等比较新的主机。

  • 【Mac 玩游戏】以太模拟器现已推出 Apple Silicon Mac 版本 PS2 模拟器

    PS 2 模拟器就用它，流畅。

  • RPCS3 - The PlayStation 3 Emulator

    PS3 模拟器，今年更新了支持 M1 的版本。尚未尝试。

  • SWITCH EMULATION on Mac! Horizon OS gaming progress on M1/M2 Apple Silicon - …

    还没在 M1 MacBook 上尝试过玩 Switch 游戏，大家可以看看这个视频，实现方式挺曲折的。

云游戏

忘了提，其实还有腾讯 START、微软 XGPU 等云游戏服务，不用占太多硬盘空间，对于普遍硬盘空间狭小的 MacBook 而言，也是比较合适的。

只不过，我一般处于一个网络条件比较差的地方，也就没有太多深入尝试。
