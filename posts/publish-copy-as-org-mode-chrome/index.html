作为 Emacs 用户一直很羡慕 markdown 有很多丰富的插件，将网页直接保存为 md 文件，在朋友 J 的鼓励下，我开发了 Copy as Org-mode Chrome，解决了长久以来的困扰

February 18, 2025
now

项目地址：https://github.com/yibie/Copy-as-org-mode-chrome

  • 直接将网页里的文本，直接转换成 org-mode 格式，拷贝到剪贴板
  • 将网页直接保存为 org-mode 格式的文件下载到你指定的目录
  • 将网址直接转换成 org-mode 格式并拷贝到剪贴板

该项目基于 kuanyui 的 Copy as Org-mode 开发。
